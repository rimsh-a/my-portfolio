# Rimsha Ijaz — Portfolio

A responsive personal portfolio built with plain HTML, CSS and JavaScript.

## Files
- `index.html` — page structure (About, Skills, Projects, Resume, Contact)
- `styles.css` — all styling, layout and animation
- `script.js` — scroll reveal, mobile nav, scroll progress bar, contact form

## Before you deploy — personalize it
1. **Contact section** (`index.html`): replace the placeholder email, GitHub and LinkedIn links.
2. **Resume button**: add your real `resume.pdf` to the folder and update the `href` on the Download Resume button.
3. **Project links**: swap the `#` placeholders under each project card for your live demo and GitHub repo links.
4. **Contact form**: it's frontend-only right now (it just shows a confirmation message). Connect it to [Formspree](https://formspree.io) or [EmailJS](https://www.emailjs.com/) so messages actually reach your inbox.
5. **Skills / bio copy**: edit the About and Skills sections to match your real experience.

## Deploy on GitHub Pages
1. Create a new GitHub repository and push these files to it.
2. Go to **Settings → Pages**.
3. Under "Build and deployment", set Source to `Deploy from a branch`, branch `main`, folder `/root`.
4. Save — your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Deploy on Netlify
1. Go to [netlify.com](https://netlify.com) and log in.
2. Drag and drop this folder onto the Netlify dashboard (Sites → "Add new site" → "Deploy manually").
3. Netlify gives you a live URL instantly — you can rename it under Site settings → Domain management.
